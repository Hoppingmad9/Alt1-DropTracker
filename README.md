# DropChecker

Tracks golden loot beams and checks if your legendary pet picks them up and alerts you if it doesn't.

Retains data until you hit the "Clear" button.

## Installation:
Install by copying this either copy pasting the following link in your browser. 

alt1://addapp/hoppingmad9.github.io/Alt1-DropTracker/appconfig.json

Or, by going to the following link in the Alt1 Browser

https://hoppingmad9.github.io/Alt1-DropTracker/

## Usage:
    1. Open app with RuneScape window visible.

    2. Ensure that the chat filter is off on the chat window you want to track drops in.  

        2a. Ensure timestamps are on!  Otherwise, the tracker may miss a drop!

    3. Make sure your transparency is turned off (Not see-through) on your interfaces!  
      
        3a. Settings -> UI Customisation -> Move Transparency slider all the way to the left.
    
    4. Go kill stuff.

    5. You can tweak the delay for the alert (time it waits for pet pick up message), time between drop alerts, and the alert volume in settings.
